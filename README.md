# Shopify CLI Theme

This repository contains a custom Shopify theme built using Shopify CLI. It is based on the Horizon/Dawn structure with custom modifications for performance, UI enhancements, and custom sections.

---

## 🚀 Project Overview

This Shopify theme includes:
- Custom sections and templates
- Updated assets (CSS, JS)
- Optimized Liquid structure
- SEO-friendly layout
- Responsive design for all devices
- Custom product, collection, and homepage layouts

---

## 📁 Folder Structure

- `assets/` → CSS, JS, images
- `config/` → Theme settings and schema configuration
- `layout/` → Main layout files (theme.liquid)
- `sections/` → Custom Shopify sections
- `snippets/` → Reusable Liquid components
- `templates/` → Page templates (product, collection, index, etc.)
- `locales/` → Language translation files
- `seed-data.js` → Sample or initial data setup
